TROUBLESHOOTING
===============

Problem: "Failed to fetch"
Solution:
- Check Apache is running in XAMPP
- Verify files are in C:\xampp\htdocs\rest-api\
- Try: http://localhost/rest-api/test-connection.php

Problem: "Database connection failed"
Solution:
- Check MySQL is running in XAMPP
- Import setup.sql in phpMyAdmin
- Check config.php has correct credentials

Problem: "Invalid credentials"
Solution:
- Make sure you imported setup.sql
- Use: admin@example.com / admin123
- Check admins table exists in database

Problem: 404 Not Found
Solution:
- Check .htaccess file exists
- Enable mod_rewrite in Apache
- Restart Apache after changes
